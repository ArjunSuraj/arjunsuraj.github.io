## Overview

Managed comprehensive end-to-end supply chain applications for Trimble's forestry sector, covering five distinct applications that handle everything from resource planning to logistics tracking.

## Technical Stack

- **Frontend:** Angular with component libraries for consistent UX across all five applications
- **Backend:** Spring Boot microservices with RESTful APIs
- **Infrastructure:** Google Cloud Platform with containerized deployment
- **Data:** Relational databases with event-driven architecture using Kafka
