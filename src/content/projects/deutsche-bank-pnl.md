## Overview

Developed Profit and Loss (P&L) web applications for Deutsche Bank during my time at GFT Technologies in the UK. The system processed real-time financial data with robust API integrations.

## Technical Stack

- **Frontend:** Angular 7 with real-time data visualization
- **Backend:** Spring Boot with SOAP/REST web services for consuming financial details
- **Infrastructure:** Google Cloud Platform with Splunk for comprehensive logging
- **Methodology:** Agile scrum with regular client presentations

## Key Contributions

- Implemented SOAP/REST web services for robust financial data integration
- Built real-time P&L dashboards for trading desk visibility
- Deployed on GCP with comprehensive monitoring and alerting
- Collaborated directly with Deutsche Bank stakeholders on business logic
